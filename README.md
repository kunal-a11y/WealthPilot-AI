## 🚀 Overview

WealthPilot AI is a premium, full-stack financial technology application designed to help users track their net worth, manage global assets, and receive real-time, AI-driven financial advice. Built with a stunning dark-mode aesthetic and glassmorphism UI, it bridges the gap between complex market data and intuitive personal finance management.

## ✨ Key Features

- **Real-Time Global Market Data:** Fetches live pricing, historical charts, and trending assets for stocks, ETFs, mutual funds, and crypto directly via a Yahoo Finance integration.
- **AI Financial Advisor:** Powered by Google Gemini, the AI acts as your personal wealth manager—analyzing your portfolio, suggesting assets, and generating deep-dive reports.
- **Dynamic Portfolio Builder:** Add, edit, and track your assets (equities, real estate, crypto) and liabilities (loans, mortgages) to calculate your exact net worth in real-time.
- **Global Market Screener:** Filter and discover top-performing assets from the US, India (NSE/BSE), Europe, Japan, and more.
- **Secure Authentication:** Enterprise-grade security and user management powered by Clerk.
- **Fully Responsive Design:** A flawless user experience across mobile phones, tablets, and desktop computers with a sleek slide-out navigation drawer.

## 🛠️ Technology Stack

- **Frontend:** React 18, TypeScript, TailwindCSS, Framer Motion, Recharts
- **Backend:** Node.js, Express, TypeScript, Prisma ORM
- **Database:** PostgreSQL
- **Authentication:** Clerk
- **AI Integration:** Google Gemini Pro
- **Market Data:** Yahoo Finance API
